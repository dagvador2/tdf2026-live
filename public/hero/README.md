# Vidéos du hero

Déposer ici :
- `hero-home.mp4` — vidéo du hero de la page d'accueil (H.264, muette, ~1080p, < 8 Mo idéalement)
- `hero-home-poster.jpg` — image affichée pendant le chargement (optionnel)

Tant que ces fichiers sont absents, la home affiche automatiquement le fond animé de secours.
